#NotebookLM Desktop (Unofficial)
A lightweight, cross-platform desktop wrapper for Google NotebookLM, built with Electron. This app brings your AI-powered research assistant out of the browser and onto your desktop with a dedicated window and native integration for Windows and Linux.

**Disclaimer: This is an unofficial community project and is not affiliated with, or endorsed by, Google. All rights to NotebookLM belong to Google.**

✨ Features
Dedicated Window: Keep your research separate from your messy browser tabs.

Platform: Native installer for Windows (.exe), But you can build it yourself.

Performance: Low-overhead Electron wrapper focused on the NotebookLM web experience.

🚀 Installation
Go to the [Releases page](https://github.com/wexloth-original/notebooklm-windowsapp-unofficial/releases).

Download the installer.

Run the installer and log in with your Google account.

And here you are.

🛠️ Development
If you want to build from source:

```
git clone https://github.com/wexloth-original/notebooklm-windowsapp-unofficial.git
cd notebooklm-windowsapp-unofficial
npm install
npm start
```
